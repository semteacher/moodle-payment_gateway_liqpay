The LiqPay plugin allows you to receive payments via LiqPay.
Working since Moodle 3.10
See https://docs.moodle.org/310/en/Payment_gateways for more

--------------------------------------------------------------------------------
To Install:

1. Load the questionnaire module directory into your "payment/gateway" subdirectory.
2. Visit your admin page to create all of the necessary data tables.

--------------------------------------------------------------------------------
To Upgrade:

1. Copy all of the files into your 'payment/gateway/liqpay' directory.
2. Visit your admin page. The database will be updated.
